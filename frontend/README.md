# portik frontend (Next.js)

**Disclaimer:** This platform does not provide investment advice. Analytics are for educational purposes only.

## Quickstart

```bash
cd frontend
cp env.example .env.local
npm install
npm run dev
```

Set `NEXT_PUBLIC_API_BASE_URL` to your backend (default: `http://localhost:8000`).

## Hard rule

No analytics / P&L calculations live in the frontend. The UI only renders backend-provided metrics.
